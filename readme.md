# Non-official Copic Color Library

Array of color objects

```javascript
const colors = require('copic-colors');
```

```javascript
{
	code: 'B39',
	name: 'Prussian Blue',
	family: 'B',
	saturation: '3',
	brightness: '9',
	hex: '23224a',
	rgb: [35, 34, 74],
	cmyk: [53, 54, 0, 71]
}
```
